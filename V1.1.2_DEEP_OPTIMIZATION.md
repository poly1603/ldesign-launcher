### 1. `src/utils/warning-suppressor.ts` (250+ 行)
**功能**: 统一的警告管理系统

**特性**:
- ✅ 自定义过滤规则（黑名单/白名单）
- ✅ 临时抑制和自动恢复
- ✅ 全局单例模式
- ✅ 内存泄漏防护
- ✅ 抑制统计功能
- ✅ Debug 模式显示被抑制的警告

**使用方式**:
```typescript
import { getGlobalSuppressor } from './utils/warning-suppressor'

// 激活全局警告抑制
const suppressor = getGlobalSuppressor()
suppressor.activate()

// 执行可能产生警告的操作
await someOperation()

// 停用警告抑制（可选，通常由系统管理）
// suppressor.deactivate()
```

## 🔨 技术实现

### 警告抑制系统

```typescript
// 全局警告抑制器
class WarningSuppressor {
  // 过滤规则
  - Node.js ESM 警告
  - Vite CJS API 警告
  - 浏览器兼容性警告
  - Sourcemap 警告
  - 实验性功能警告
  
  // 核心方法
  - activate() - 激活抑制
  - deactivate() - 停用抑制
  - withSuppression() - 临时抑制
  - addFilter() - 添加过滤规则
}
```
